🎮 Hangman Game - Python
📌 Project Overview

This project is a simple text-based Hangman Game developed in Python as part of my CodeAlpha Internship.

The player guesses a hidden word one letter at a time. The game provides six chances for incorrect guesses before the player loses.

🚀 Features
Random word selection
5 predefined words
Letter-by-letter guessing
Input validation
Tracks incorrect guesses
Win/Loss detection
Beginner-friendly Python implementation
🛠 Technologies Used
Python 3
Random Module
Lists
Strings
While Loops
Conditional Statements

▶ How to Run
Install Python 3.
Download or clone this repository.
Open terminal in project folder.
Run:
python hangman.py
